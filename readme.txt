                             
                                                   Teste-T
    
    
  Teste-T é um programa feito em perl usado para calcular o desvio padrão, a média e o Teste-T  entre duas amostras. Para fazer o programa foi estudado conceitos sobre estatística, teste de médias, desvio padrão e variância.
   Criado por Júlia Castro, Daniel Morais e Francislon em 2016.
   Ao iniciar o programa deve se inserir um conjunto de réplicas, é calculado o desvio padrão e a média e então é inserido o segundo conjunto de réplicas onde são calculados os mesmos atributos. Após obter o segundo conjunto o programa calcula desvio padrão e o Teste-T entre as amostras.
